# Python Automation Toolkit

A practical automation toolkit built with Python for **web scraping, browser automation, and productivity tasks**.
This project demonstrates automation skills using **Selenium WebDriver, BeautifulSoup4, Requests, and Git**.

## 🚀 Project Highlights

* Built a **web scraping module** using BeautifulSoup4 + Requests for automated HTML/XML data extraction.
* Developed a **Selenium WebDriver automation script** for end-to-end UI testing.
* Implemented reusable Python scripts for automation workflows.
* Managed project source code using **Git & GitHub**.
* Focused on clean, modular, and scalable scripting practices.

## 🛠️ Tech Stack

* Python 3.x
* Selenium WebDriver
* BeautifulSoup4
* Requests
* lxml / html.parser
* Git & GitHub
